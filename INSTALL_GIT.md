# Установка Git для Windows

## Быстрая установка

### Способ 1: Через winget (рекомендуется)

Откройте PowerShell или командную строку и выполните:

```powershell
winget install Git.Git
```

### Способ 2: Через официальный сайт

1. Перейдите на https://git-scm.com/download/win
2. Скачайте установщик (автоматически определит 64-bit версию)
3. Запустите установщик
4. Используйте настройки по умолчанию (можно оставить все галочки)
5. Выберите редактор по умолчанию (рекомендуется VS Code или Notepad++)

### Способ 3: Через Chocolatey

Если у вас установлен Chocolatey:

```powershell
choco install git
```

---

## Проверка установки

После установки откройте новое окно PowerShell или командной строки и выполните:

```bash
git --version
```

Должна отобразиться версия Git, например: `git version 2.42.0`

---

## Первоначальная настройка

После установки Git выполните следующие команды:

```bash
# Настройте ваше имя (замените на ваше реальное имя)
git config --global user.name "Ваше Имя"

# Настройте ваш email
git config --global user.email "ваш@email.com"

# Настройте окончания строк для Windows
git config --global core.autocrlf true

# Включите цветной вывод
git config --global color.ui auto

# Настройте редактор по умолчанию (если используете VS Code)
git config --global core.editor "code --wait"
```

## Инициализация репозитория в текущем проекте

После установки и настройки Git выполните в корне проекта:

```bash
# Инициализация репозитория
git init

# Добавление всех файлов
git add .

# Первый коммит
git commit -m "Initial commit: структура проекта и документация"

# Создание основной ветки
git branch -M main
```

## Следующие шаги

1. **Создайте аккаунт на GitLab:**
   - Перейдите на https://gitlab.com
   - Или используйте собственный инстанс GitLab

2. **Создайте новый проект** в GitLab

3. **Подключите удалённый репозиторий:**
   ```bash
   git remote add origin https://gitlab.com/ваш-username/vet-assistant.git
   git push -u origin main
   ```
