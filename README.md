# Ассистент ветеринара

Мобильное приложение для ветеринарных врачей, позволяющее создавать протоколы осмотра животных с помощью голосового ввода.

## Описание

Приложение позволяет ветеринарным врачам:
- Записывать голосовые сообщения с описанием осмотра
- Автоматически распознавать речь и формировать протоколы
- Редактировать протоколы вручную
- Сохранять протоколы в PDF формате
- Работать офлайн

## Технологии

- Flutter (Dart)
- SQLite (Drift)
- Speech-to-Text (гибридный режим: облако + офлайн)
- PDF генерация

## Структура проекта

```
lib/
├── core/                    # Общие утилиты, константы, DI
├── features/
│   ├── patients/            # Модуль пациентов
│   ├── templates/           # Модуль шаблонов протоколов
│   ├── examinations/        # Модуль осмотров/протоколов
│   ├── speech/              # Модуль распознавания речи
│   ├── pdf/                 # Модуль генерации PDF
│   └── admin/               # Административный модуль
└── main.dart
```

## Установка

1. Установите Flutter SDK
2. Установите зависимости:
```bash
flutter pub get
```

3. Запустите приложение:
```bash
flutter run
```
